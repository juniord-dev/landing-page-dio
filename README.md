# Desafio: Criando sua primeira Landing Page com HTML e CSS

Esse projeto foi desenvolvido a partir da Trilha de CSS da DIO! Trata-se de uma Landing Page com HTML e CSS, colocando em prática os fundamentos do CSS,
as propriedades básicas da linguagem de estilização, além de trabalhar com as unidades de medidas relativas e absolutas.
